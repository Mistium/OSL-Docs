# canvas

```javascript
import "osl/canvas"
```

## Methods

- `canvas.width()` → `number`
- `canvas.height()` → `number`
- `canvas.pixels()` → `number`
- `canvas.setPixel(idx, hexColor)`
- `canvas.getPixel(idx)` → `string`
- `canvas.setPixelAt(x, y, hexColor)`
- `canvas.getPixelAt(x, y)` → `string`
- `canvas.fill(hexColor)`
- `canvas.clear()`
- `canvas.stretch(newW, newH)`
- `canvas.toURL()` → `string`
- `canvas.toArr()` → `array`
