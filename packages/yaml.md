# yaml

> YAML parsing and encoding utilities

```javascript
import "osl/yaml"
```

## Methods

- `yaml.parse(source)` Ôćĺ `any`
- `yaml.stringify(data)` Ôćĺ `string`
- `yaml.toJSON(yamlData)` Ôćĺ `string`
- `yaml.fromJSON(jsonData)` Ôćĺ `any`
- `yaml.get(data, key)` Ôćĺ `any`
- `yaml.set(data, key, value)` Ôćĺ `object`
- `yaml.merge(data1, data2)` Ôćĺ `object`
- `yaml.keys(data)` Ôćĺ `array`
- `yaml.values(data)` Ôćĺ `array`
- `yaml.has(data, key)` Ôćĺ `boolean`
- `yaml.delete(data, key)` Ôćĺ `object`
