# xml

> XML parsing and manipulation utilities

```javascript
import "osl/xml"
```

## Methods

- `xml.toStr()` Ôćĺ `string`
- `xml.toArr()` Ôćĺ `array`
- `xml.findNode(path)` Ôćĺ `xmlNode`
- `xml.getText(path)` Ôćĺ `any`
- `xml.getAttr(path, attr)` Ôćĺ `any`
- `xml.get(path)` Ôćĺ `object`
- `xml.getAll(path)` Ôćĺ `array`
- `xml.has(path)` Ôćĺ `boolean`
- `xml.hasAttr(path, attr)` Ôćĺ `boolean`
- `xml.setText(path, value)`
- `xml.setAttr(path, attr, value)`
- `xml.count(path)` Ôćĺ `number`
- `xml.remove(path)`
- `xml.clear(path)`
