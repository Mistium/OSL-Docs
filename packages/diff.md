# diff

> File and text diff utilities for comparing differences

```javascript
import "osl/diff"
```

## Methods

- `diff.text(oldStr, newStr)` → `DiffResult`
- `diff.words(oldStr, newStr)` → `DiffResult`
- `diff.chars(oldStr, newStr)` → `DiffResult`
- `diff.unified(result)` → `string`
- `diff.html(result)` → `string`
- `diff.json(result)` → `string`
