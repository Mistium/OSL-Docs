# random

> Seedable pseudo-random utilities (fast, deterministic when seeded)

```javascript
import "osl/random"
```

## Methods

- `random.seed(n)`
- `random.float(...args)` Ôćĺ `number`
- `random.int(...args)` Ôćĺ `number`
- `random.between(min, max)` Ôćĺ `number`
- `random.bool(...args)` Ôćĺ `boolean`
- `random.pick(arr)` Ôćĺ `any`
- `random.shuffle(arr)` Ôćĺ `array`
- `random.sample(arr, n)` Ôćĺ `array`
- `random.string(...args)` Ôćĺ `string`
- `random.gaussian(...args)` Ôćĺ `number`
