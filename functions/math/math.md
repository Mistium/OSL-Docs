# math - Package, not a function

> ⚠️ **`Math()` is not a callable function.**

`math` is a **package**, not a builtin function. To use math functions and constants, import the package first:

```javascript
import "math"

log math.PI
// 3.141592653589793

angle = math.sin(0.5)
```

For details on available math functions and constants, see the [math package documentation](../../packages/math.md).
