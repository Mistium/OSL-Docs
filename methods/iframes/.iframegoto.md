# .iframeGoto(x, y)

## Description

`iframeGoto` moves an iframe to a position on screen, in the same coordinate space
as the [draw cursor](../../commands/rendering/draw-cursor/README.md). Pair it with
[`.iframeResize()`](.iframeresize.md) to lay the embed out.

## Parameters

- `x`: the horizontal position to move the iframe to.
- `y`: the vertical position to move the iframe to.

## Usage

```javascript
frame = "https://example.com".iframeNew()

frame.iframeGoto(100, 50)
// positions the top-left of the iframe at (100, 50)
```
