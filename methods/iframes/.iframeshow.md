# .iframeShow()

## Description

`iframeShow` makes a hidden iframe visible again. It is the counterpart to
[`.iframeHide()`](.iframehide.md).

## Parameters

`iframeShow` takes no parameters; it is called on an iframe id returned by
[`.iframeNew()`](.iframenew.md).

## Usage

```javascript
frame = "https://example.com".iframeNew()

frame.iframeHide()
// ... later ...
frame.iframeShow()
// the iframe is visible again
```
