# .iframeClose()

## Description

`iframeClose` destroys an iframe and frees its resources. After closing, the iframe
id is no longer valid and cannot be shown again — create a new one with
[`.iframeNew()`](.iframenew.md) if needed.

## Parameters

`iframeClose` takes no parameters; it is called on an iframe id returned by
[`.iframeNew()`](.iframenew.md).

## Usage

```javascript
frame = "https://example.com".iframeNew()

frame.iframeClose()
// the iframe is removed and its id is no longer usable
```
