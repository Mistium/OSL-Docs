# .getType()

## Description

getType returns the data type of its input value

Learn more about types here: [types.md](../../basics/types.md "mention")

## Parameters

getType takes no parameters

## Usage

```javascript
val = 16

log val.getType()
// "number"
```

```javascript
val = true

log val.getType()
// "boolean"
```

```javascript
val = "hello"

log val.getType()
// "string"
```

```javascript
val = []

log val.getType()
// "array"
```

```javascript
val = {}

log val.getType()
// "object"
```
