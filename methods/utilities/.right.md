# .right(num)

## Description

Right allows you to take part of an array or a string from the right

## Parameters

Right takes one parameter, the number of letters or items to take

## Usage On Arrays

```javascript
log [1,3,2,4].right(2)
// [2,4]
```

## Usage On Strings

```javascript
log 1324.right(2)
// 24
```
