# .last()

## Description

This method simply returns the last item of a string or an array

## Parameters

Last takes no parameters

## Usage On Arrays

```javascript
arr = [1,2,3,4,5]
// setup the array

log arr.last()
// 5
```

## Usage On Strings

```javascript
str = "12345"
// setup the string

log str.last()
// 5
```
