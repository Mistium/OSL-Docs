# .JsonParse()

> ⚠️ **Not a method in the current OSL CLI.** Use the [`json` package](../../packages/json.md) instead: `json.parse()`.

## Description

When used on an object, returns the object.

## Parameters

`JsonParse` takes no parameters.

## Usage

```javascript
obj @= {a:1,b:2,c:3}
// setup the object

log obj.JsonParse()
// returns {a:1,b:2,c:3}
// returns the object
```
