# .append(val)

> ⚠️ **Not a number method.** `.append()` works only on strings. See [strings/.append.md](/methods/strings/.append.md).

## Description

Treats the **number** as a string, appends `val`, and returns the resulting **string**.

## Parameters

* **val** — value to append (converted to string first).

## Usage

```javascript
log 42.append("px") // "42px"
```
