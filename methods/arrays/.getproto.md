# .getProto()

## Description

`getProto` returns the internal prototype object for the current array instance.
This is mostly useful for reflection or when you need to inspect what methods /
properties exist on the prototype at runtime.

## Parameters

`getProto` takes no parameters.

## Usage On Arrays

```javascript
proto @= [1, 2, 3].getProto()
log proto
// { "map": function, "filter": function, ... }
```
