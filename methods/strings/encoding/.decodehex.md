# .decodeHex()

## Description

decodeHex takes a hexadecimal representation of text, generated by [.encodehex.md](.encodehex.md "mention") and converts it back to a string

## Parameters

decodeHex takes no parameters

## Usage

```javascript
val = "68656c6c6f20776f726c64"

log val.decodeHex()
// hello world
```
