# .decodeBin()

## Description

decodeBin takes each section of binary and converts it into a character

## Parameters

decodeBin takes no parameters

## Usage

```javascript
val = "1101000 1100101 1101100 1101100 1101111 100000 1110111 1101111 1110010 1101100 1100100"

log val.decodeBin()
// hello world
```
