# .hashSHA1()

## Description

hashSHA1 simply returns its input value as an sha1 hash

{% embed url="https://www.wikiwand.com/en/articles/SHA1" %}

## Parameters

hashSHA1 takes no parameters

## Usage

```javascript
val = "hello world"

log val.hashSHA1()
// 2aae6c35c94fcfb415dbe95f408b9ce91ee846ed
```
