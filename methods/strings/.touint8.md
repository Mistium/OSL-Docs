# .toUint8()

## Description

toUint8 converts a string to an array of UTF-8 byte values.

## Parameters

toUint8 takes no parameters.

## Usage On Strings

```javascript
str = "Hello"
log str.toUint8()
// ["72", "101", "108", "108", "111"]
// Returns array of UTF-8 byte values

str = "ðŸŒŸ"
log str.toUint8()
// ["240", "159", "140", "159"]
// Unicode characters may use multiple bytes

str = ""
log str.toUint8()
// []
// Empty string returns empty array
```
