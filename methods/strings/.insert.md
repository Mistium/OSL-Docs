# .insert(location,val)

## Description

Insert is a method that puts a string into a string at a location

## Parameters

Insert takes the location to insert a string and the string to insert as parameters.

## Usage On Strings

```javascript
str = "hello world"
// setup the string

log str.insert(6," im")
// returns "hello im world"
// "im" was inserted at index 6
```
