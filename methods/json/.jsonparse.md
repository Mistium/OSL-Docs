# .JsonParse()

> ⚠️ **Not type methods.** These are functions of the [`json` package](../../packages/json.md): use `json.parse()`, `json.stringify()`, `json.format()`, `json.clean()`.

## Description

Converts a JSON string into a JS object

## Parameters

JsonParse takes no parameters

## Usage On Arrays/Objects

```javascript
arr = "[\"wow\"]"
// setup the string

log arr.JsonParse()
// returns array ["wow"]
```
