# .JsonFormat()

> ⚠️ **Not type methods.** These are functions of the [`json` package](../../packages/json.md): use `json.parse()`, `json.stringify()`, `json.format()`, `json.clean()`.

## Description

Converts A JS object or JSON string into a formatted JSON string

## Parameters

JsonFormat takes no parameters

## Usage On Arrays/Objects

```javascript
arr = ["wow","crazy"]
// setup the array

log arr.JsonFormat()
// returns string 
// [
//   "wow",
//   "crazy"
// ]
```
