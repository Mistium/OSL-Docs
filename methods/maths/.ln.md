# .ln()

## Description

This method is simply a wrapper for JavaScript's Math.log()

{% embed url="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/log" %}

## Parameters

Log takes no parameters

## Usage On Numbers

```javascript
log -1.ln()
// NaN

log -0.ln()
// -Infinity

log 0.ln()
// -Infinity

log 1.ln()
// 0

log 10.ln()
// 2.302585092994046
```
