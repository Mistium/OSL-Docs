# Files

> **Legacy (originOS).** This describes the OFSF file model used by originOS OSL.
> The OSL CLI uses the host filesystem through the [`fs`](../../packages/fs.md)
> package instead. These pages are kept for reference.

OFSF (Origin File System Format) was the in-environment file model. The pages below
cover how files were represented and the commands used to read and list them.
